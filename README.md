# Idaho Landscape Assessment Tool

Professional Progressive Web App (PWA) for field-based property assessments and landscape evaluations.

## Overview

This tool enables landscaping crews to conduct comprehensive property assessments on tablets in the field. It features an intelligent plant database specific to Idaho conditions, automatic prioritization of maintenance needs, and generates professional PDF reports for clients.

## Features

### Current Version: v4.5

- **126-Plant Idaho Database** (95 native, 31 premium cultivars)
- **Smart Prioritization** - Automatic priority assignment based on plant conditions
- **Genus-Based Intelligence** - Season-specific care recommendations for 6 plant families
- **Photo Documentation** - Capture and embed photos in assessments
- **PDF Report Generation** - Professional client-ready reports
- **Offline Capable** - PWA technology works without internet connection

## Database Coverage

**Native Problem-Solvers:**
- Screening plants (Ninebark, Blue Elderberry, Sumacs)
- Foundation shrubs (Winterfat, Creeping Oregon Grape)
- Native grasses (Idaho Fescue, Bluebunch Wheatgrass)
- Shade trees (Blue Spruce, Western White Pine)
- Perennials (Penstemon, Wild Blue Flax, Blanket Flower)

**Premium Showstoppers:**
- Japanese Maples (Bloodgood, Emperor I) - Zone 5 hardy
- Specialty Conifers (Chief Joseph Pine, Weeping Blue Atlas Cedar)
- Flowering Trees (Chanticleer Pear, Autumn Blaze Maple)
- Ornamental Interest (Cardinal Royal Mountain Ash)

## Usage

**For Field Crews:**
1. Open URL on tablet (add to home screen for quick access)
2. Enter client property information
3. Walk property documenting plants by area
4. Take photos of problem areas
5. System auto-assigns priority levels
6. Generate and share PDF report

**For Homeowners:**
- Use as reference for plant care timing
- Understand maintenance requirements
- Get Idaho-specific growing information

## Technical Details

- **Technology:** Progressive Web App (HTML5, JavaScript, CSS)
- **Compatibility:** Modern browsers (Chrome, Safari, Firefox, Edge)
- **Mobile-First:** Optimized for tablets and smartphones
- **Camera Access:** Uses device camera for photo documentation
- **PDF Generation:** Client-side PDF creation (no server required)
